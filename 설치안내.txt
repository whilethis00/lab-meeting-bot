========================================================
         랩미팅 봇 설치 안내 (개인 PC 설치용)
========================================================

이 봇은 클로바노트로 기록한 회의록을 자동으로 요약하고
할 일을 추출해주는 텔레그램 봇입니다.

준비물: Windows PC, 인터넷 연결

--------------------------------------------------------
STEP 1. Python 설치
--------------------------------------------------------

  1. 아래 주소에서 Python 설치 프로그램 다운로드
     https://www.python.org/downloads/
     → "Download Python 3.11.x" 버튼 클릭

  2. 설치 실행 시 중요:
     ☑ "Add Python to PATH" 반드시 체크 후 Install Now 클릭

  3. 설치 완료 후 확인:
     시작 메뉴 → 검색 → "cmd" 실행 → 아래 입력
     python --version
     → Python 3.11.x 이 보이면 성공

--------------------------------------------------------
STEP 2. 텔레그램 봇 만들기 (봇 토큰 발급)
--------------------------------------------------------

  1. 텔레그램 앱에서 @BotFather 검색 → 대화 시작
  2. /newbot 입력
  3. 봇 이름 입력 (예: 우리연구실 회의봇)
  4. 봇 username 입력 (영어, 반드시 bot으로 끝나야 함)
     예: mylab_meeting_bot
  5. 완료 후 "Use this token to access the HTTP API:" 아래
     긴 문자열이 봇 토큰입니다. 복사해두세요.

     예시: 1234567890:AAECf4de4o2pOTnS-6jAkBShZaiW0FrUg5Q

  ※ 토큰은 비밀번호와 같습니다. 외부에 공유하지 마세요.

--------------------------------------------------------
STEP 3. Claude API 키 발급
--------------------------------------------------------

  1. https://console.anthropic.com 접속 → 계정 생성 또는 로그인
  2. 왼쪽 메뉴 → "API Keys" → "Create Key" 클릭
  3. 이름 설정 후 생성 → 키 복사 (sk-ant-... 로 시작)

  ※ API 사용은 소액의 요금이 발생합니다.
     회의록 1건 처리 시 약 $0.002 ~ $0.005 수준입니다.
     사용량은 console.anthropic.com 에서 확인 가능합니다.
     연구실에서 하나의 키를 공유해도 됩니다.

--------------------------------------------------------
STEP 4. 봇 설치
--------------------------------------------------------

  1. 받은 압축파일(lab-meeting-bot.zip)을 원하는 위치에 압축 해제
     예: C:\lab-meeting-bot\

  2. 폴더 안의 setup.bat 더블클릭
     → Python 패키지 자동 설치
     → 설치 완료 후 .env 파일이 자동으로 메모장에서 열립니다

  3. 메모장에서 아래 두 줄을 수정하고 저장 (Ctrl+S):

     TELEGRAM_BOT_TOKEN=여기에_텔레그램_봇_토큰_붙여넣기
     ANTHROPIC_API_KEY=여기에_ANTHROPIC_API_키_붙여넣기

     → "여기에_..." 부분을 STEP 2, 3에서 복사한 값으로 교체
     → 주의: = 앞뒤에 공백 없이 붙여쓰기

     예시:
     TELEGRAM_BOT_TOKEN=1234567890:AAECf4de4o2pOTnS-6jAkBShZaiW0FrUg5Q
     ANTHROPIC_API_KEY=sk-ant-api03-abcdef1234567890...

--------------------------------------------------------
STEP 5. 봇 실행
--------------------------------------------------------

  1. start_bot.bat 더블클릭
  2. "랩미팅 봇이 실행되었습니다!" 메시지 확인
  3. 텔레그램에서 본인 봇 username 검색 (예: @mylab_meeting_bot)
  4. /start 입력 → 정상 작동 확인

  봇 종료: stop_bot.bat 더블클릭
  봇 재시작: start_bot.bat 더블클릭 (종료 후 자동 재시작)

  ※ 봇은 PC가 켜져 있는 동안만 작동합니다.
     PC를 껐다 켠 후에는 start_bot.bat 을 다시 실행해주세요.

--------------------------------------------------------
봇 사용법
--------------------------------------------------------

[ 회의록 분석 ]

  방법 1 - 텍스트 붙여넣기:
    클로바노트 앱 → 해당 회의 → 공유 → 텍스트 복사
    → 텔레그램 채팅창에 붙여넣기 후 전송

  방법 2 - 파일 첨부:
    클로바노트에서 .txt 파일로 저장 → 텔레그램 파일 첨부로 전송

[ 화자 이름 설정 ]
  클로바노트는 "화자 1", "화자 2" 로 기록됩니다.
  실제 이름으로 바꾸려면:

  "화자 1은 김연구원이고 화자 2는 박연구원이야"
  또는 /setname 화자1 김연구원

  → 한 번 설정하면 이후 회의록에 자동 적용됩니다.

[ 명령어 ]
  /start     봇 소개
  /help      도움말
  /tasks     미완료 할 일 목록
  /meetings  최근 회의 목록
  /names     화자 이름 설정 현황
  /setname   화자 이름 설정

[ 자연어 질문 ]
  "지난주 미팅에서 결정된 거 뭐야?"
  "이번 달 할 일 보여줘"
  "실험 B 관련 내용 찾아줘"

--------------------------------------------------------
문제 해결
--------------------------------------------------------

  Q. setup.bat 실행 시 "Python이 설치되어 있지 않습니다" 오류
  A. STEP 1로 돌아가서 Python 재설치. "Add to PATH" 체크 필수.

  Q. start_bot.bat 실행 후 텔레그램에서 봇이 응답 없음
  A. .env 파일의 TELEGRAM_BOT_TOKEN 값을 다시 확인해주세요.
     봇 토큰에 공백이 포함되지 않았는지 확인.

  Q. 회의록 분석 시 "오류가 발생했습니다" 메시지
  A. .env 파일의 ANTHROPIC_API_KEY 값을 다시 확인해주세요.
     API 크레딧이 충분한지 console.anthropic.com 에서 확인.

  Q. 봇이 갑자기 응답하지 않음
  A. stop_bot.bat 실행 후 start_bot.bat 재실행.
